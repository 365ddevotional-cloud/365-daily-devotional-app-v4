STEP 2 — Apply Random Background

In the devotional greeting card component add:

import { devotionalBackgrounds } from "@/constants/devotionalBackgrounds"

const randomBackground =
devotionalBackgrounds[Math.floor(Math.random() * devotionalBackgrounds.length)]


STEP 3 — Apply Background Styling

Add to the card container:

style={{
backgroundImage: `url(${randomBackground})`,
backgroundSize: "cover",
backgroundPosition: "center"
}}


STEP 4 — Add Dark Overlay for Readability

Add overlay layer:

background: rgba(0,0,0,0.45)
